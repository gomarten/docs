# Marten Documentation

Documentation site for [Marten](https://github.com/gomarten/marten) - the Go web framework you reach for when you want nothing in the way.

## Local Development

### Prerequisites

- Python 3.8+
- pip

### Setup

```bash
# Install dependencies
pip install mkdocs-material mkdocs-minify-plugin

# Start dev server
mkdocs serve
```

Visit `http://localhost:8000`

### Build

```bash
mkdocs build
```

Output goes to `site/` directory.
